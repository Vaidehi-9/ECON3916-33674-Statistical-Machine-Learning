# ECON 3916 Final Project — Student Performance Prediction
 
**Author:** Vaidehi Pundeer  
**Course:** ECON 3916, Spring 2026  
**Dataset:** UCI Student Performance (Math), accessed April 2026
 
## Prediction Question
Can we predict whether a secondary school student will fail their final math exam using only demographic, social, and behavioral information available at the start of the school year?
 
## Data Source
- **Dataset:** UCI Student Performance Dataset
- **DOI:** https://doi.org/10.24432/C5TG7T
- **Loaded from:** https://quantdev.ssri.psu.edu/sites/qdev/files/student-mat.csv

## How to Run
1. Open `econ3916_final.ipynb` in Google Colab or Jupyter
2. Run all cells top to bottom (Runtime → Run all in Colab)
3. No local data download needed — data loads directly from URL in the notebook
